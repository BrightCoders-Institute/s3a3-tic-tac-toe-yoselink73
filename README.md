## Descripcion
este proyecto es el desarrollo del tipico juego Tic Tac Toe tambien conocido como (Gato) donde en un tablero de 3x3 dos jugadores se turnan para ocupar cada casilla con un simbolo que puede ser "X" o "O" donde el ganador es quien haga de manera horizontal, vertical o diagonal 3 simbolos iguales.

## Como instalar y correr el proyecto

para correr este proyecto antes es necesario:

- tener instalado ruby
- clonar el repositorio de Github
- cambiarse a la carpeta donde fue clonado
- desde esa carpeta correr la clase main.rb
